# Project: auto-signal-bot

## Stack
- Runtime: Node.js 20+
- Language: TypeScript (strict mode)
- Package manager: npm
- Test runner: Vitest (tests/ at project root)
- Build: tsc (`npm run build`)
- Entry: src/scripts/index.ts (có thể thay đổi — kiểm tra package.json)

## Key Files & Directories

| Path | Vai trò |
|------|---------|
| `src/` | Source code chính |
| `src/betting/` | Betting logic, odds parsing, signal generation |
| `src/charts/` | Chart analysis, verify provider |
| `src/scripts/` | Scripts entry points |
| `src/config/` | Configuration (JSON files) |
| `tests/` | Test files, mirror src/ structure |
| `supabase/` | Database migrations, Supabase config |
| `tasks/` | Multi-agent task queue (protocol files) |
| `CLAUDE.md` | Protocol for Claude Code Desktop (auto-load) |
| `AGENTS.md` | Protocol for Hermes profiles |
| `.project-summary.md` | (this file) Project context for agents |

## Conventions
- Error handling: return `new Error(...)` objects, không throw
- Arrow functions cho callbacks
- Config: JSON files trong `src/config/`
- Testing: Vitest, mirror src/ structure

## Multi-Agent Protocol
- File-based task queue tại `tasks/`
- Lead (planner): tạo plan.md + task.md
- Worker (executor): thực thi task.md, viết result.md
- Workflow: Plan → Deploy task files → Worker execute → Lead review
- Chi tiết: xem `CLAUDE.md` hoặc `AGENTS.md`
